Assumptions: [All element nodes have an id,
	      That means all parent nodes have an id,
	      All element nodes follow "~" scheme for id if appropriate]
If anchor node has an id:
   If anchor node has any element node children:
      Base newSpan id off the appropriate child's.
   Else:
      Base newSpan id off the anchor node's.
Else:
   If anchor node's parent node has any element node children:
      Base newSpan id off the appropriate child's.
   Else:
      Base newSpan id off the anchor node's parent node's.
